# PaiCoop Dev Docs

> Development Documents for PaiCoop

## Ussage


### Install Node.js

安装Node.js到电脑。Win10参考[这篇博客](https://segmentfault.com/a/1190000038318834).


### Install Dependencies

```shell
$ npm install
```

### Local Browse

```shell
$ npm run serve
```

Use browser to visit `http://localhost:3000`
